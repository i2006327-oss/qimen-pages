# 奇門遁甲純網頁版部署說明

這個資料夾可以直接上傳到 GitHub Pages、Cloudflare Pages、Netlify，或任何靜態網頁空間。

## 入口

- 首頁：`index.html`
- 不需要 Python
- 不需要 exe
- 不需要後端 API

## 上傳方式

1. 將本資料夾內全部檔案上傳到網站根目錄。
2. 確認網站會開啟 `index.html`。
3. 用 iPhone 或 Android 開啟網站網址即可使用。

## 手機使用

- 建議用 HTTPS 網址。
- 第一次開啟需要網路。
- 開啟後會註冊 PWA 快取，之後可加入主畫面並支援離線使用。

## 注意

不要只把 zip 傳到 iPhone 開啟。iOS 不能把 zip 當網站執行，必須先上傳到 GitHub Pages / Cloudflare Pages 這類靜態網站服務，或在電腦解壓後用瀏覽器打開。
